# Automated AWS PostgreSQL Deployment
This project uses **Terraform** to provision a RHEL 9 server on AWS and **Ansible** to automatically install and secure a PostgreSQL database with real-time monitoring.

## Features
- Infrastructure as Code (Terraform)
- Automated Configuration (Ansible)
- Security Hardening (PostgreSQL access control)
- Monitoring (Node Exporter for CPU/RAM)# aws_ansible_postgres_project
